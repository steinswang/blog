## 说明
### 搜索框
| Key | Descption |
| :- | :- |
| ESC | 1.如果输入框有内容，清除内容<br>2.如果输入框无内容，失去焦点 |
| i/I | 获取焦点 |
| 下 | 向下选择文章 |
| 上 | 向上选择文章 |
| 回车 | 打开当前选中的文章，若没有，则默认打开第一个 |

### 全局
| Key | Descption |
| :- | :- |
| s/S | 全屏/取消全屏 |
| w/W | 打开/关闭文章目录 |
| j/J | 向下滑动 |
| k/K | 向上滑动 |
| gg/GG | 到最顶端 |
| shift+G/g | 到最下端 |
